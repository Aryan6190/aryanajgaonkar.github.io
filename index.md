# Real-Time Thief Detection System
Real-time computer vision-based object detection system that flags suspicious movement using Python and OpenCV.

Project Overview-
This project is a computer vision-based real-time thief detection system built using Python and OpenCV. 
It processes live video feeds to detect suspicious movement and classify objects accurately. 
The system improves surveillance automation and provides a foundation for real-time alerting workflows.

Technologies used-
Language: Python
Libraries/Frameworks: OpenCV, NumPy
Methodologies: Computer Vision, CNN (Convolutional Neural Networks), Haar Cascade
Tools: VSCode, Git
